# zipica-home

A simple static website for Zipica.

## Files

- `index.html`: Main homepage
- `terms-of-service.html`: Terms of service page
- `public/`: Directory for additional public assets

## Running Locally

To serve the HTML files locally, use the provided script:

```bash
./serve.sh
```

This starts a Python HTTP server on port 8000. Visit http://localhost:8000 to view the site.

Alternatively, run directly:

```bash
python3 -m http.server 8000
```
